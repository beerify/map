## Development Notes

### Install Expo
npm install -g expo

### Get the code
`git pull https://github.com/beerify/map.git`

## Install dependencies
Do this only if you wish to run the code

`cd map && npm install`

## Start the App
### Android
`expo start:android`

or

`expo start` then press a

### iOS
`expo start:ios`

or

`expo start` then press i

### Web
`expo start:web`

or

`expo start` then press w
